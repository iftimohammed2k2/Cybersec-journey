# John the Ripper Mini Project

A simple demonstration of password cracking using John the Ripper and a custom wordlist.

## Project Files

- `sample_hashes.txt`: Contains simulated MD5 hashes.
- `mywordlist.txt`: Custom dictionary with common passwords.
- `crack.sh`: Script to run John and display cracked passwords.
